# College Course Recommendation System (JavaFX + Spring Boot + MySQL + LLM Agent)

A full-stack Java application that recommends college courses based on a student's **major**, **interests**, and **completed courses**.  
It includes a **JavaFX desktop planner UI**, a **Spring Boot REST API**, a **MySQL database**, and a **real LLM agent** that generates explainable recommendations.

---

## Features
- ✅ Student profiles (major, year, interests, completed courses)
- ✅ Course catalog with tags + prerequisites
- ✅ Rule-based filtering + scoring (prereq-safe)
- ✅ LLM agent adds human-like explanations (server-side, secure)
- ✅ JavaFX semester planner UI (add/remove recommended courses)
- ✅ MySQL persistence + automatic demo data seeding

---

## Tech Stack
- **Frontend:** JavaFX (Desktop)
- **Backend:** Spring Boot (REST API)
- **Database:** MySQL
- **AI Agent:** OpenAI Responses API (server-side)
